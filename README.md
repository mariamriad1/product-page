# DJI Mavic 4 Pro Product Page

A responsive product landing page for the DJI Mavic 4 Pro drone, featuring clean layouts, comprehensive technical specifications, user reviews, and subscription forms.

## Features

* Modern and clean UI style
* Responsive product specifications sections
* Flexbox layouts
* Semantic HTML structure

## Tech Stack

* HTML5
* CSS3

Clone the repository:

```
git clone https://github.com/mariamriad1/product-page.git
```
